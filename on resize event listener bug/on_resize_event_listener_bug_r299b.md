## Problem description

I posted the HTML5 canvas. When changing the full-screen mode, it does not have time to settle down in the correct position on the screen.

## Attach a .c3p

[on_resize_event_listener_bug_r299b.zip](https://github.com/WilsonPercival/WilsonPercival/files/8980615/on_resize_event_listener_bug_r299b.zip)

## Steps to reproduce

1. Open and run the project.
2. Switch to full screen mode and windowed mode.
