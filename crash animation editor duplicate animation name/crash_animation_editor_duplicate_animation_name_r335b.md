## Problem description

Construct crashes.

## Attach a .c3p

[crash_animation_editor_duplicate_animation_name_r335b.zip](https://github.com/WilsonPercival/WilsonPercival/files/11028270/crash_animation_editor_duplicate_animation_name_r335b.zip)

## Steps to reproduce

1. Open a project.
2. Open the image editor.
3. Click on `New folder`.
4. Press `Ctrl + C`.
5. Press quickly `Ctrl + V` until you see a crash.
