## Problem description

When I change the value of the `Angle` parameter in the left ear to `0.01`, it is rounded to `0°`.

## Attach a .c3p

[round_angle_bug_r364b.zip](https://github.com/WilsonPercival/WilsonPercival/files/13192645/round_angle_bug_r364b.zip)

## Steps to reproduce

1. Open a project.
2. Click on the sprite.
3. In the left ear, change the value of the `Angle` parameter to `0.01`.
