## Problem description

When I'm in mobile view and I press the mouse buttons in a specific order, a bunch of errors are generated in the console.

## Attach a .c3p

[request_fullscreen_error_r311b.zip](https://github.com/WilsonPercival/WilsonPercival/files/9641252/request_fullscreen_error_r311b.zip)

## Steps to reproduce

1. Open and run the project.
2. Open the console.
3. Click on the `Toggle device toolbar` button.
4. Follow the pattern of clicking the mouse buttons - you need to click quickly and rhythmically. I demonstrated it in the gif below.

![pattern](https://user-images.githubusercontent.com/91274932/192154440-88a57925-826e-4aab-8088-79a81be64be5.gif)

## Observed result

![observed](https://user-images.githubusercontent.com/91274932/192154432-1d74b311-4181-422a-b436-1f95fdbf59c5.gif)

## Expected result

The console does not generate errors.
