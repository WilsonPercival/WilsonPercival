## Problem description

When I change the device orientation from landscape to portrait, the game does not display correctly.

## Attach a .c3p

[change_screen_orientation_bug_r302s.zip](https://github.com/WilsonPercival/WilsonPercival/files/9123458/change_screen_orientation_bug_r302s.zip)

## Steps to reproduce

1. Create a new project.
2. Resize the viewport to 720x1280.
3. Set the full screen mode to `Scale outer`.
4. Add sprites to the layout.
5. Launch via remote preview.
6. Hold your device in landscape orientation.
7. When the game loads - change the device orientation to landscape.
8. You will see an incorrect display.
9. Reload the page.
10. You see the correct display.
