## Problem description

The canvas does not paste the sprite if there is a `Color` effect on the sprite.

## Attach a .c3p

[drawing_canvas_color_effect_paste_bug_r331b.zip](https://github.com/WilsonPercival/WilsonPercival/files/10846113/drawing_canvas_color_effect_paste_bug_r331b.zip)

## Steps to reproduce

1. Open and run the project.
2. Move the sprite.
